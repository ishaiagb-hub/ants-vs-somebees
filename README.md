# Ants vs. SomeBees
This project implements core game logic for a turn-based simulation where ants defend against invading bees, inspired by Plants vs. Zombies.

## Features (In Progress)
- Object-oriented design using inheritance and class hierarchies  
- Agent-based interactions between ants and bees  
- Turn-based game logic and state updates  

## Current Progress
- Implemented core gameplay mechanics for basic ant types (HarvesterAnt, ThrowerAnt)  
- Built foundational class structure for Places, Insects, and game state  

## Files
- `ants.py` – main game logic and class implementations  

## Skills
Python, object-oriented programming, data structures, simulation
